# Тетрис — консольный тетрис на C

Классическая игра «Тетрис», написанная на C для терминала с использованием ncurses.

## Возможности

- ✅ Классический игровой процесс с 7 стандартными фигурами
- ✅ Управление с клавиатуры в реальном времени (стрелки, вращение, жёсткое падение)
- ✅ Подсчёт очков и отображение уровня
- ✅ Интерфейс на основе ncurses
- ✅ Модульные тесты с использованием check

## Зависимости

- **gcc** — компилятор C
- **make** — система сборки
- **ncurses** — отрисовка интерфейса в терминале
- **check** — библиотека для модульного тестирования

### Установка на Linux (Ubuntu/Debian)

sudo apt update
sudo apt install gcc make libncurses-dev check

### Установка на Windows (MSYS2)

1. Скачайте и установите MSYS2
2. Откройте терминал **MSYS2 UCRT64** и выполните:

pacman -Syu
pacman -S mingw-w64-ucrt-x86_64-gcc make mingw-w64-ucrt-x86_64-ncurses mingw-w64-ucrt-x86_64-check


## Сборка и запуск

### Сборка игры

make

### Запуск игры

make run

### Запуск тестов

make test

## Поддерживаемые платформы

| Платформа | Статус | Примечания |
| :--- | :--- | :--- |
| Linux | ✅ Полная поддержка | GCC + ncurses из коробки |
| Windows | ⚠️ Через MSYS2 | См. инструкцию по установке выше |
| macOS | ❓ Не тестировалось | Должно работать с `brew install ncurses check` |
